# lldDiagrams

This folder contains low-level design (LLD) diagrams for the LLD-Design-Patterns repository. Store UML diagrams (class diagrams, sequence diagrams, state diagrams, component diagrams) and any other design artifacts here.

Preferred file formats:
- .drawio (editable)
- .png / .svg (exported for easy viewing)
- .pdf

Recommended structure:
- lldDiagrams/<pattern-name>.<drawio|png|svg|pdf>
- lldDiagrams/README.md (this file)

Contributing:
1. Add source (.drawio) files whenever possible so diagrams are editable.
2. Include exported images (.png or .svg) for quick preview.
3. Name files using the pattern name, e.g. factory-method.drawio, singleton.png
